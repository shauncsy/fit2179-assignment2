NBL vs NBA Basketball Visualisation

This data visualisation project compares the Australian National Basketball League with the NBA, focusing on teams, locations, player performance, and league development.

Project Introduction

Basketball is becoming increasingly popular in Australia. This project uses interactive visualisations to explore how the NBL compares with the NBA in terms of geography, team structure, and player performance.

Visualisation 1: NBL Team Locations

This map shows NBL team locations across Australia and New Zealand using the prepared NBL team statistics dataset. The number inside each point represents the team's calculated ranking, while the colour represents win percentage. Users can hover over each point to view team performance details such as wins, losses, shooting percentage, rebounds, assists, and turnovers.
